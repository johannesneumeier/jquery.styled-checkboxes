Include jquery, jquery.styled-checkboxes.js and your theme css

<script type="text/javascript" src="path/to/jquery-1.8.1.min.js"></script> <script type="text/javascript" src="path/to/jquery.styled-checkboxes/jquery.styled-checkboxes-0.1.js"></script> <link rel="stylesheet" href="path/to/jquery.styled-checkboxes/themes/raw/jquery.styled-checkboxes.css" />

Example form html

<form> <label for="a-checkbox">Normal checkbox:</label> <input name="a-checkbox" type="checkbox" /> <br/> <label for="some-checkbox">Checked checkbox:</label> <input name="some-checkbox" type="checkbox" checked="checked" /> <br/> <label> Checkbox within a label <input type="checkbox" /> </label> </form>

Transform the checkboxes like this

<script type="text/javascript"> $(document).ready(function () { $('form input[type="checkbox"]').styledCheckboxes(); }); </script>
