jQuery.styled-checkboxes examples

Basic setup

Include jquery, jquery.styled-checkboxes.js and your theme css

<script type="text/javascript" src="PATH/TO/jquery-1.8.1.min.js"></script> <script type="text/javascript" src="PATH/TO/jquery.styled-checkboxes/jquery.styled-checkboxes-0.1.js"></script> <link rel="stylesheet" href="PATH/TO/jquery.styled-checkboxes/themes/raw/jquery.styled-checkboxes.css" />

The included raw theme is a very stripped-down version that let's you easily modify the theme to include background-images or other css markup to customize your own theme

Example form html

<form> <label for="a-checkbox">Normal checkbox:</label> <input name="a-checkbox" type="checkbox" /> <br/> <label for="some-checkbox">Checked checkbox:</label> <input name="some-checkbox" type="checkbox" checked="checked" /> <br/> <label> Checkbox within a label <input type="checkbox" /> </label> </form>

Transform the checkboxes like this

<script type="text/javascript"> $(document).ready(function () { $('form input[type="checkbox"]').styledCheckboxes(); }); </script>


Customized setup

Like in the basic setup, include jquery, jquery.styled-checkboxes, and link to your customized css

<script type="text/javascript" src="PATH/TO/jquery-1.8.1.min.js"></script> <script type="text/javascript" src="PATH/TO/jquery.styled-checkboxes/jquery.styled-checkboxes-0.1.js"></script> <link rel="stylesheet" href="PATH/TO/jquery.styled-checkboxes/themes/YOURTHEME/jquery.styled-checkboxes.css" />

Then setup jquery.styled-checkboxes with custom settings

In this instance, the default wrapping element was changed to a be a span with class .custom-checkbox. Also, the default value for 'tick' was overwritten from 'false' to 'span', which means that there is an additional element to show the checked status inside the wrapper, which you can style seperately.

<script type="text/javascript"> $(document).ready(function () { $('form.example-default input[type="checkbox"]').styledCheckboxes(); $('form.example-custom input[type="checkbox"]').styledCheckboxes({ 'wrapper': 'span', 'wrapperClass': 'custom-checkbox', 'tick': 'span' }); }); </script>


List of options you can modify

checkedClass
class that the wrapper of a checked checkbox has, like so: 'checkedClass' : 'your-css-class' (notice, no . in front of the css class),
defaults to checked
tick
if you want an additional html element inside the checkbox wrapper, set it here like so: 'tick' : 'span',
defaults to false
tickClass
class of the additional tick html element, set it like this: 'tickClass' : 'your-css-class' (notice, no . in front of the css class),
defaults to styled-checkboxes-tick
tickContent
if you want any content inserted in the tick, you could add an 'x'-letter like this: 'tickContent' : 'x'. you can set this content to be hidden when a checkbox is not checked by using css.
defaults to &nbsp; (resulting in an empty space)
wrapper
use this to set what html element is being used to wrap the checkbox, use it like so: 'wrapper' : 'span',
defaults to div
wrapperClass
the css class given to the wrapping html element, customize it by using: 'wrapperClass' : 'your css class' (again, the css class without a .).
defaults to styled-checkboxes
